# Sistem Monitoring Getaran Turbin Berbasis Rust

## Deskripsi Project
Project ini adalah sistem simulasi monitoring getaran (vibration monitoring) pada turbin untuk mendeteksi kondisi Normal, Warning, atau Bahaya secara real-time. Program ini menggunakan bahasa Rust untuk menjamin keamanan memori dan performa tinggi, serta menggunakan metode Moving Average untuk pengolahan data.

## Cara Menjalankan Program
1. Pastikan Anda sudah menginstal Rust (Cargo).
2. Masuk ke direktori project: `cd src`
3. Jalankan program dengan perintah:
   ```bash
   cargo run
